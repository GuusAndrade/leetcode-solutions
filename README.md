# LeetCode monorepo (Flutter + Java)

This repository is a monorepo containing:

- `flutter/`: Flutter project used to practice LeetCode problems in **Dart** (with unit tests).
- `java/`: Java solutions.

## Working on Dart/Flutter solutions

Dart solutions live under:

- `flutter/lib/leetcode/`

And Dart tests under:

- `flutter/test/leetcode/`

### Run tests

```bash
cd flutter
flutter test
```

### Create a new problem

Example for `valid_parentheses`:

- `flutter/lib/leetcode/valid_parentheses.dart`
- `flutter/test/leetcode/valid_parentheses_test.dart`
